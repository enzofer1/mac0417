As informações sobre as imagens estão contidas no próprio nome delas, no formato:

cgfah_diaOUnoite_FUNDO_dentroOUfora_repeticao 
OU
exsbf_diaOUnoite_FUNDO_dentroOUfora_repeticao

cgfah significa colher,garfo,fio dental, abridor, hashi -> objetos que estão na imagem
exsbf significa esponja, caneca, sabonete, bomba para mate, faca -> objetos que estão na imagem.

diaOUnoite ->  D se a foto foi tirada de dia e N caso tenha sido tirada de noite
FUNDO -> B(branco) ou P(preto) ou V(verde)
dentroOUfora -> D se foi tirada dentro de casa ou F se foi tirada fora de casa
repeticao -> 1 se foi a primeira imagem e 2 e 3 se forem a segunda ou terceira repetição;

as imagens que representam as classes colher,garfo,fio dental, abridor e hashi SÃO AS MESMAS 
as imagens que representam as classes esponja, caneca, sabonete, bomba para mate e faca SÃO AS MESMAS
